# Dream Café

A minimal, editorial website for **Dream Café** — specialty coffee, pastries, and light fare across Sofia (Bulgaria) and Greece.

**Live:** deploy this repository to your static host of choice (see [TECHNICAL.md](./TECHNICAL.md) for deployment and stack details).

## Quick start

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080`.
